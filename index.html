Application Overview

AeroMatch is an interactive web app that helps aviation enthusiasts and students discover their ideal aircraft, flight path, or aviation career match through a fun, personality-based quiz. From future pilots to aerospace engineers, users answer a series of questions and instantly see their personalized results.

Technology Overview

AeroMatch will be developed using HTML, CSS, and JavaScript for the front-end interface. The application will use the JavaScript framework to create an interactive, component-based quiz system and dynamic result rendering. For styling, CSS will streamline the design process, providing responsive layouts and modern UI consistency. The site will be hosted on GitHub Pages during development and testing, with deployment automation managed via GitHub Actions.
